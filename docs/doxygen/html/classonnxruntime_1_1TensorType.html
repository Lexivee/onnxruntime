Detailed Description

template<typename elemT>
class onnxruntime::TensorType< elemT >

Tensor type. This type does not have a C++ type associated with it at registration time except the element type. One of the types mentioned above at IsTensorContainedType<> list is acceptable.

Usage: ONNXRUNTIME_REGISTER_TENSOR(ELEMENT_TYPE) Currently all of the Tensors irrespective of the dimensions are mapped to Tensor<type> type. IsCompatible() currently ignores shape.
