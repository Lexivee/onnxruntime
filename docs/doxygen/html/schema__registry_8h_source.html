64 // OnnxRuntimeOpSchemaRegistry is used to provide supplement for built-in ONNX schemas.
65 // Each OnnxRuntimeOpSchemaRegistry must register complete opsets delta from a baseline version to max opset version.
66 // (Please notice that baseline opsets are not include in the delta)
67 // For example, ONNXRuntime is build with ONNX 1.2 which is at opset7, to use onnx opset8 and opset9,
68 // user could create a OnnxRuntimeOpSchemaRegistry and config it as {baseline_opset_version = 7, opset_version = 9}
69 // it means this OnnxRuntimeOpSchemaRegistry contains the complete delta from opset7 to opset9.
