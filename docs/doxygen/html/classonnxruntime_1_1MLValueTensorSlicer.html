Detailed Description

template<typename T>
class onnxruntime::MLValueTensorSlicer< T >

Class to provide a slicing service over a Tensor stored within an MLValue with shape {batch size, sequence length, <input shape>="">}. Access to the slices is via an iterator interface.

For each iteration an MLValue will be returned containing a sub-Tensor of the original Tensor. The sub-Tensor applies the relevant offset to the data address from the original Tensor in order to avoid any memory allocations/copies for the tensor data.
